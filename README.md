# Tai Ahom Revival

A web-based digital platform for preservation and revival of Tai Ahom civilization.

## Features
- History & Dynasties
- Language & Script
- Weapons & Warfare
- Culture & Lifestyle
- Heritage Sites
- Research & Manuscripts
- Admin-controlled content

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB

## Purpose
- B.Tech Final Year Project
- Long-term public cultural platform
